# 介绍

提示

页面模板源码免费开源，在uni-app的插件市场uview-plus的 示例项目 中，在右上角选择"使用 HBuilderX 导入示例项目" 或者 "下载示例项目ZIP"， 在HX运行项目即可看到和使用模板。

布局为uview-plus框架的一部分，是uview-plus平时积累的常见页面布局，可能是页面的一部分，可也能是整个 页面，主要功能不是逻辑的交互，而是样式的实现，所以我们把它们收集成一个板块，而不是当成组件呈现。

需要使用这些布局的用户，可以直接复制到需要的页面，修改相关的结构，逻辑或者样式即可。

注意

这些布局都依赖uview-plus框架，请确保在使用uview-plus的情况下使用相关的功能，否则，需要自行调试修改。
